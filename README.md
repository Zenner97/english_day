# Мой день — тренажёр английского

Тренажёр устной речи на тему «мой день / планы на день». Построен по методу:
**Список → Слушать → Повторять (шэдоуинг) → Вспомнить (active recall) → Рассказать.**
Контент — реальный день ученицы (по её голосовому), расширенный до ~38 фраз: прошедшее время (рассказ про вчера), настоящее (обычный день), будущее (планы) и фразы-связки для плавности.

Один файл `index.html`, без сборки и зависимостей. Прогресс хранится на устройстве ученицы.

---

## Быстрый старт

Просто открой `index.html` в браузере. Озвучка сразу работает через встроенный голос браузера (Web Speech API) — этого достаточно, чтобы заниматься уже сегодня.

Чтобы звук был приятнее (голос ElevenLabs), сгенерируй mp3 один раз — см. ниже. Тренажёр сам подхватит файлы из папки `audio/`, а если их нет — мягко откатится на голос браузера.

## Озвучка через ElevenLabs (один раз)

Так сделано специально: ключ остаётся у тебя на машине и **не попадает в репозиторий**, и нет ошибки `Failed to fetch` (её вызывает CORS при вызове ElevenLabs прямо из браузера). Скрипт генерит mp3 заранее, ты их коммитишь, сайт играет статику.

Нужен Node.js 18+.

```bash
cd day-trainer
ELEVENLABS_API_KEY=sk_твой_ключ node generate_audio.mjs
```

Голос уже прописан в скрипте по умолчанию (`VOICE_ID=aKUMgdkpitgitOAQ9gZN`). Создастся папка `audio/` с файлами `01.mp3 … 38.mp3`.

Необязательные настройки (через переменные окружения):

```bash
# перезаписать уже существующие файлы / сменить голос или модель
FORCE=1 ELEVENLABS_API_KEY=sk_... node generate_audio.mjs
VOICE_ID=другой_id MODEL_ID=eleven_turbo_v2_5 ELEVENLABS_API_KEY=sk_... node generate_audio.mjs
```

Ключ передаётся только через переменную окружения — **не вписывай его в файлы и не коммить.**

Стоимость — копеечная: это разовая генерация ~38 коротких фраз. Ученица потом слушает уже готовые файлы сколько угодно, без обращений к API.

## Публикация на GitHub Pages

1. Создай репозиторий и положи туда `index.html`, `generate_audio.mjs`, `README.md` и папку `audio/` (с уже сгенерированными mp3).
2. Запушь:
   ```bash
   git init && git add . && git commit -m "Day trainer"
   git branch -M main
   git remote add origin https://github.com/USERNAME/REPO.git
   git push -u origin main
   ```
3. В репозитории: **Settings → Pages → Source: Deploy from a branch → main / (root) → Save.**
4. Через минуту сайт будет по адресу `https://USERNAME.github.io/REPO/`. Эту ссылку и отправь ученице.

`.gitignore` не нужен — ключа в файлах нет (он передаётся только через переменную окружения). Если вдруг Pages не отдаёт что-то из папки — добавь в корень пустой файл `.nojekyll`.

## Как ученице заниматься (у неё один день)

Открыть с телефона, сверху есть блок «Как заниматься» с планом. Коротко:

- **Список** — прочитать все фразы, нажать на любую, чтобы услышать.
- **Слушать** — включить «Слушать всё» фоном (в дороге, за делами), 20–30 минут. Цель: чтобы звук фразы стал родным.
- **Повторять** — говорить вслух вместе с диктором, в том же темпе. Есть замедление 0,75× и (где поддерживается) запись себя для сравнения.
- **Вспомнить** — видит русское, говорит английское сама, потом проверяет. «Вспомнила» ставит зелёную галочку, «Ещё раз» возвращает фразу в круг. Полоса прогресса сверху растёт.
- **Рассказать** — три задания (вчерашний день / обычное утро / планы на сегодня) с подсказками-связками, чек-листом пунктов и таймером. Это и есть то, что она покажет на уроке.

Рекомендация: к уроку пройти «Вспомнить» до зелёных галочек и проговорить «Рассказать» 2–3 раза подряд (последний — без подсказок).

## Поменять контент

Все фразы — в массиве `ITEMS` в `index.html`. Те же `id` и тексты продублированы в `generate_audio.mjs` (имя файла = `id`). Если меняешь/добавляешь фразы — поправь оба места одинаково и перегенерируй озвучку.

Совместимость: iOS Safari, Android, десктоп. Звук стартует по тапу (требование iOS), используется один переиспользуемый аудио-элемент.
